## Directories ##
* 000 = Neutral Zones
* 001 = Missions/Basic Maps
* 002 = Endless Maps
* 003 = Challenge Maps
* 004 = Raid Maps


## Sub-Directories ##
Area-{ID}
Area-0001 contains files for the first map created

ID is used in Lua and .dbr


## Files ##

a001 is the first region of the first variation of the map
